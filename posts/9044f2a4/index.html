<a></a>
作用：

  • 另一个文档
  • 文档内部
  • Email地址
  • 电话号码

属性

  • href
  • target
  • download
  • hreflang
  • rel
  • rev
  • type
  • referrerpolicy

target中属性值

  • _self
  • _blank
  • _parent
  • _top

举例：
创建锚点
<a href="#">×××</a>
链接Email地址
<a href="mailto:1078857679@qq.com">×××</a>
链接电话
<a href="tel:15651005256">×××</a>
文件下载
<a href="/keyboard-shortcuts.pdf" download>×××</a>
