class类

发表于2022-09-22|更新于2022-10-18
|字数总计:1.4k|阅读时长:5分钟

在es5中主要是通过构造函数方式和原型方式来定义一个类
在es6中可以通过class来定义类。
一个类只能拥有一个名为”constructor”构造函数，不能出现多个；
如果没有定义”constructor”构造函数，class会默认添加一个空的“constructor”构造函数。

class类必须进行new调用，不能直接执行

使用new调用类的构造函数会执行如下操作。

  1. 在内存中创建一个新对象；
  2. 这个新对象内部的[[Prototype]]指针被赋值为构造函数的prototype属性；
  3. 构造函数内部的this被赋值为这个新对象（即this指向新对象）；
  4. 执行构造函数内部的代码（给新对象添加属性）；
  5. 如果构造函数返回非空对象，则返回该对象；否则，返回刚创建的新对象；

class类不存在变量提升

class类受块作用域限制

class类无法遍历其实例原型链上的属性和方法

不可枚举

new.target属性

es6为new命令引入了一个new.target属性，它会返回new命令作用于的那个构造函数。
如果不是通过new调用或Reflect.construct()调用的，new.target会返回undefined

class类有static静态方法

static静态方法只能通过类调用，不会出现在实例上；
静态方法包含 this 关键字，这个 this 指的是类，而不是实例
static声明的静态属性和方法都可以被子类继承。

静态属性

静态属性指的是类本身的属性，而不是定义在实例对象this上的属性。

静态方法

可以在类里面定义静态方法，该方法不会被对象实例继承，而是直接通过类来调用。

静态方法里使用this是指向类。

关于方法的调用范围限制，比如：私有公有，ES6暂时没有提供，一般是通过约定，比如：在方法前面加下划线_print()表示私有方法。

getter 与 setter

在 class 内部可以使用 get 与 set 关键字，对某个属性设置存值函数和取值函数，拦截该属性的存取行为。

constructor()

constructor()方法是类的默认方法，new生成实例对象时会自动调用该方法。

一个类必须有constructor()方法，如果没有显式定义，引擎会默认添加一个空的constructor()。

constructor()方法默认返回实例对象（即this）。

getter和setter

与 ES5 一样，在类的内部可以使用get和set关键字，对某个属性设置存值函数和取值函数，拦截该属性的存取行为。

this

类的方法内部的this，它默认指向类的实例，在调用存在this的方法时，需要使用 obj.method()方式，否则会报错。

如果要单独调用又不报错，一种方法可以在构造方法里调用bind(this)。

bind(this) 会创建一个新函数，并将传入的this作为该函数在调用时上下文指向。

另外可以使用箭头函数，因为箭头函数内部的this总是指向定义时所在的对象。
