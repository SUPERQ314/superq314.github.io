reflect反射机制

发表于2022-09-23|更新于2022-10-18
|字数总计:656|阅读时长:2分钟

MDN定义：

Reflect是一个内置的对象，用来提供方法去拦截JavaScript的操作。Reflect不是一个函数对象，所以它是不可构造的，也就是说它不是一个构造器，你不能通过 new操作符去新建或者将其作为一个函数去调用Reflect对象。Reflect的所有属性和方法都是静态的。

反射机制指的是程序在运行时能够获取自身的信息。js 中的 apply 就是反射机制。

Reflect API 汇总
Reflect 提供了一套用于操作对象的 API（静态方法）

handler⽅法 默认设置调⽤ 作用
get Reflect.get(target, name, receiver) 获得目标的身上某一特性的值
set Reflect.set(target, name, value, receiver) 在目标上设定特性
has Reflect.has(obj, name) 分辨一个目标是不是存有某一特性
deleteProperty Reflect.deleteProperty(obj, name) 删掉目标上的特性
getProperty Reflect.getPrototypeOf(obj) 获得特定目标原形的涵数
setProperty Reflect.setPrototypeOf(obj, newProto) 设定或更改目标原形的涵数
isExtensible Reflect.isExtensible(target) 分辨一个目标是不是可拓展（即是不是可以加上新的特性）
preventExtensions Reflect.preventExtensions(target) 阻拦新特性加上到目标
getOwnPropertyDescriptor Reflect.getOwnPropertyDescriptor(target, propertyKey) 获得给出特性的特性描述符
defineProperty Reflect.defineProperty(target, propertyKey, attributes) 界定或改动一个目标的特性
ownKeys Reflect.ownKeys() 回到由总体目标目标本身的特性键构成的二维数组
apply Reflect.apply(func, thisArg, args) 对一个涵数开展启用实际操作，另外能够传到一个二维数组做为启用主要参数
construct Reflect.construct(target, args) 对构造方法开展new实际操作，完成建立类的案例
preventExtensions Reflect.preventExtensions() 阻拦新特性加上到目标

Reflect对象的作用/优点

  1. 把实现反射机制的方法重新归结在一起并且简化操作，保持JS语意清晰和语法简单
  1. 补充一些Object对象没有的方法（如Reflect.apply）
  1. 让Object操作都变为函数行为，比如使用Reflect.has(obj, name)替换name in obj
