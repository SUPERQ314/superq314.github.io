JavaScript学习笔记

JavaScript内置对象

内置对象就是js语言自带的一些对象，这些对象供开发者使用，并提供了一些常用的或是最基本而必要的功能（属性和方法）

内置对象最大的优点是帮助程序员快速开发

JavaScript提供的内置对象：Number、Math、Date、Array、String、RegExp、Global、Error

标准内置对象的分类

详细资料可以参考：

Number 数值对象

Number - JavaScript | MDN (mozilla.org)

JavaScript 的 Number 对象是经过封装的能让你处理数字值的对象。Number 对象由 Number() 构造器创建。

JavaScript 的 Number类型为双精度 IEEE 754 64 位浮点类型。

col1 col2
var num = new Number(10);
typeof num		 “Object”
Number(“1.1”) 1.1
Number(0xA) 10
Number(true) 1
Number(“12.a”) NaN

epsilon
两个可表示 (representable) 数之间的最小间隔。

max_safe_integer
JavaScript 中最大的安全整数 (2^53 - 1)。

max_value
能表示的最大正数。最小的负数是 -MAX_VALUE。

min_safe_integer
JavaScript 中最小的安全整数 (-(2^53 - 1)).

min_value
能表示的最小正数即最接近 0 的正数 (实际上不会变成 0)。最大的负数是 -MIN_VALUE。

特殊的“非数字”值。

negative_infinity
特殊的负无穷大值，在溢出时返回该值。

positive_infinity
特殊的正无穷大值，在溢出时返回该值。

Number 对象上允许的额外属性。

作用：确定传递的值是否是 NaN。
Number.isNaN(value)
参数：
- value：要检测是否为 NaN 的值。
返回值：
* 布尔值，表示给定的值是否是 NaN。

作用：确定传递的值类型及本身是否是有限数。
Number.isFinite(value)
参数：
- value：要被检测有穷性的值。
返回值
* 布尔值 表示给定的值是否是一个有穷数。

作用：确定传递的值类型是“number”，且是整数。
Number.isInteger(Number)
参数：
- number：检测的值
返回值：
* 布尔值（true/false）

作用：确定传递的值是否为安全整数 ( -(2^53 - 1) 至 2^53 - 1) 之间。
Number.isSafeInteger(testValue)
参数：
- testValue：需要检测的参数。
返回值：
* 布尔值 表示给定的值是否是一个安全整数（safe integer）。

作用：和全局对象 parseFloat() 一样。
Number.parseFloat(string)
参数：
- string：被解析的字符串。
返回值
* 给定值被解析成浮点数，如果无法被解析成浮点数，则返回NaN

作用：和全局对象 parseInt() 一样。
Number.parseInt(string[, radix])
参数：
- string：要解析的值。 如果此参数不是字符串，则使用 ToString 抽象操作将其转换为字符串。忽略此参数中的前导空格。
- radix：一个介于 2 到 36 之间的整数，代表字符串的基数 (数学数字系统中的基)。
返回值
* 转换后的整型数

作用：将数值转换成指数形式，返回字符串。
numObj.toExponential(fractionDigits)
参数
- fractionDigits（可选）：整数，用来指定小数点后有几位数字。默认情况下用尽可能多的位数来显示数字。
返回值
* 使用指数形式表示的字符串
异常
* RangeError
如果 fractionDigits 太小或太大将会抛出该错误。介于 0 和 20（包括 20）之间的值不会引起 RangeError 。 执行环境也可以支持更大或更小范围。
* TypeError
如果该方法在一个非数值类型对象上调用。

作用：将数值保留指定位数的小数位。
numObj.toFixed(digits)
参数
- digits：小数点后数字的个数；介于 0 到 20（包括）之间。如果忽略该参数，则默认为 0。
返回值
* 定点表示法格式化后的字符串。

抛出异常
* RangeError
如果 digits 参数太小或太大。0 到 20（包括）之间的值不会引起 RangeError。实现环境（implementations）也可以支持更大或更小的值。
* TypeError
如果该方法在一个非Number类型的对象上调用。

作用：返回数字在特定语言环境下表示的字符串。覆盖 Object.prototype.toLocaleString() 方法。
numObj.toLocaleString([locales [, options]])
参数：
返回值
* 返回一个语言环境下的表示字符串。

作用：以指定精度返回该数值的字符串表示形式。
numObj.toPrecision(precision)
参数
- precision（可选）：一个用来指定有效数个数的整数。
返回值
* 指定位数表示的字符串。
异常
* RangeError
如果 precison 参数不在 1 和 100（包括）之间，将会抛出一个 RangeError 。执行环境也可以支持更大或更小的范围。ECMA-262 只需要最多 21 位显示数字。

作用：返回一个代表给定对象的字符串，基于指定的基数。
numObj.toString([radix])
参数：
- radix：指定要用于数字到字符串的转换的基数 (从 2 到 36)。如果未指定 radix 参数，则默认值为 10。
返回值：
* 字符串
异常信息
* RangeError
如果 toString() 的 radix 参数不在 2 到 36 之间，将会抛出一个 RangeError。

作用：返回指定对象的原始值。覆盖 Object.prototype.valueOf() 方法。
numObj.valueOf()
返回值
* 表示指定 Number 对象的原始值的数字

方法 作用 示例 返回值

charCodeAt() 返回字符串中特定索引处的字符的ASCll码 str.charCodeAt(0)
toUpperCase() 将字符串的所有字符转换成大写字母 str.toUpperCase()
toLowerCase() 将字符串的所有字符转换成小写字母 str.toLowerCase()
indexOf 返回字符串中特定字符串第一次出现的位置 str.indexOf(‘Q’, [起始位置])

只会替换第一个字符
split() 将某个字符串分割成多个字符串，可以指定分隔符 str.split(‘,’)
string.prototype.match(regexp) 使用正则表达式搜索目标子字符串 返回匹配结果的数组

trim() 移除字符串首尾空白

示例：查找字母 o ，”haeobcosndosoaoaoaoaxovfodo”

核心算法：

1、先查找第一个o出现的位置

2、indexOf()返回的结果不是-1，就继续往后查找

3、因为indexOf()只能查找到第一个就停止，所以后面的查找，需要用第二个参数的索引加1，以此类推，直到indexOf() ==-1结束

步骤：

1、核心算法：利用chartAt()遍历这个字符串

2、把每个字符都存储给对象，如果对象没有该属性，就为1，如果存在了就+1

3、遍历对象，得到最大值和该字符

RegExp 正则对象

概述

正则表达式用于定义一些字符串的规则，计算机可以根据正则表达式，来检查一个字符串是否符合规则，获取将字符串中符合规则的内容提取出来。

使用typeof检查正则对象，会返回object。

目的

  • 给定的字符串是否符合正则表达式的过滤逻辑(称作”匹配”)
  • 可以通过正则表达式，从字符串中获取想要的特定部分

正则表达式的特点

  • 灵活性、逻辑性和功能性非常的强
  • 可以迅速地用极简单的方式达到字符串的复杂控制
