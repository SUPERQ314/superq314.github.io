VScode常用扩展插件

发表于2022-06-18|更新于2022-06-28|VScode

VScode插件

主题及图标

插件名 功能
One Dark Pro VScode的最佳选择
GitHub Theme 黑白两款皮肤
Material Theme 集成了多种主题皮肤，搭配Material Icon Theme更佳
Material Icon Theme 扁平化的主题图标库
vscode-icons VScode官方图标库，为VScode资源目录树加上图标
Emoji 按f1输入emoji，可以看到有三个选项，分别是
      1、emoji表情
      2、markdown下的emoji
      3、unicode下的emoji
选中一个模式回车进入列表，再回车即可输入到代码中
:emojisense: 快捷键：ctrl + i

功能强化

插件名 功能
Chinese (Simplified) Language Pack for
Visual Studio Code		 编辑器汉化语言包
在locale.json中添加"locale": "zh-cn"，即可载入中文（简体）语言包
Code Runner 代码运行器，快捷键：Ctrl + Alt + M，一键运行多种语言运行代码片段或代码文件
Debugger for Chrome 前端调试
Image Preview 鼠标悬浮在链接上，可预览图片
Live Server 本地静态服务器
Markdown Preview Enhanced markdown编辑和预览，边编辑markdonwn边预览
Regex Previewer 预览正则表达式效果
settings sync VScode配置同步到云端，同步所有设置和插件
SVG Viewer 预览SVG文件
TODO Tree 使用TODO来标记代码，提高可读性；
提供可视化窗口来查看和管理TODO Tree
wakatime 编程时间及行为跟踪统计

Git 集成

插件名 功能
GitHub Pull requests 在VScode中查看和管理GitHub拉取请求和问题
Git Graph Git图形化显示和操作
GitHistory 可查看和搜索Git日志以及图形和详细信息；
同时还支持分支比较，分支管理等操作

编程美化

插件名 功能
Beautify 在代码文件右键鼠标一键格式化HTML、JS、CSS
Beautify css/sass/scss/less
Bracket Pair Colorizer 因为此功能现在内置于VScode，此扩展已弃用
Bracket Pair Colorizer 2 常用括号显示不同颜色，让层次结构一目了然；
还支持用户自定义符号
change-case 更改命名风格
Codelf 给变量起名的神器
Code Spell Checker 拼写检查程序，检查不常见的单词，如果单词拼写错误，会给出警告提示
Color Info 提供CSS中使用颜色的相关信息
只需在颜色上悬停光标，就可以预览色块中色彩模型的（HEX、RGB、HSL、CMYK）相关信息
Color Picker 输入pick color，调出调色板
EditorConfig for VS Code 代码风格统一
formate: CSS/LESS/SCSS formatter CSS样式美化，VScode内置CSS格式化功能，支持less、scss
Highlight Matching Tag 突出显示匹配的开始/结束标签
Minify 用于压缩合并JS和CSS文件的应用程序；<br />提供自定义设置，以及自动压缩保存并导出为.min文件的选项；<br />可分别通过uglify-js、clean-css、html-minifier与JS、CSS、HTML协同工作
Prettier - Code formatter 自动格式化代
Trailing Spaces 显示尾随空格

绘图

插件名 功能
Markmap 思维导图神器
Polacode 代码截图
Polacode-2020 可将代码转化成图片
vscode-drawio 可在VScode里画流程图
新建.drawio后缀文件并拖入VScode中，即可得到流程图界面

前端开发

插件名 功能
Auto Complate Tag 自动添加HTML/XML关闭标签
Auto Close Tag 自动添加HTML/XML关闭标签
Auto Rename Tag 自动重命名配对的HTML/XML标记，修改开始标签，结束标签会同步修改
canvas-snippets
css-auto-prefix
CSS Peek CSS样式查看器，可快速查看CSS样式，非常方便快捷
ESLint 规范js代码书写规则，可自定义规则
HTML Boilerplate 使用HTML模版，避免重写HTML头部和正文标签的苦恼。
在空文件中输入HTML，并按Tab键，即可生成干净的文档结构
HTML Snippets HTML代码片段，该插件可为你提供HTML标签的代码提示，不用键入尖括号了
JavaScript (ES6) code snippets ES6语法智能提示，以及快速输入
json2ts 自动把json格式转成ts的类型，复制json之后按ctrl + alt + v即可
koroFileHeader 自动添加头部注释和函数注释的插件；
支持自定义内容，需要在settings.json中进行自定义配置
Live Sass Compiler SASS实时编译
npm 支持运行package.json文件中定义的npm脚本
根据package.json中定义的依赖项验证已安装的模块
Npm Intellisense 可自动完成导入语句中的npm模块
open in browser 快速打开HTML文件到浏览器预览
Path Intellisense 智能路径提示，方便快速引入文件
px to rem & rpx (cssrem) 自动换算单位的插件，自动换算单位的插件
Turbo Console Log 快速添加console.log信息，JS debug必备快捷键
      1、ctrl + alt + l选中变量之后，使用这个快捷键生成console.log
      2、alt + shift + c注释所有console.log
      3、alt + shift + u启用所有console.log
      4、alt + shift + d删除所有console.log
Vetur Vue开发者必备，内含语法高亮、智能提示、emmet、错误提示、格式化、自动补全、debugger等实用功能

数据库及数据分析

插件名 功能
Data Preview 预览数据文件
MongoDB for VS Code MongoDB数据库支持
MySQL 支持MySQL/MariaDB、Microsoft SQL Server、PostgreSQL、Redis、ElasticSearch
Oracle Developer Tools for VS Code Oracle数据库支持
Sort lines 对当前文本排序
SQL Server (mssql) Microsoft SQL Server数据库支持

插件名 功能
