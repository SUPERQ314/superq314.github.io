hexo 标题与 id 关系

hexo markdown 文章里面如果使用 ## 文章二级标题 生成标题，标题会自动生成 id="文章二级标题" 。
如果标题中间有 半角空格 或者 全角空格 ，空格都会变成连字符，如：

  • 一个半角空格 ## hello world 或 多个半角 ## hello world 或 多个全角空格 ## hello　　　world 。
  • 其生成的 id 会变成 id="hello-world"，多个空格会合并成一个连字符。
